# Web приложение для приемки заказа на доставку

## Инструкция по запуску проекта:
1. Для установки нужно скачать zip архив с github репозитория. 
2. Распаковать zip архив.
3. В папке проекта найти и запустить файл "DeliveryApp.sln".  
4. После заменить строку подключения к бд ("DefaultConnection") в файлах appsettings.json на свою или настроить подключение к "mssqllocaldb".

#### P.S. Инструкцию по установке локальной БД "mssqllocaldb" можно найти на сайте: https://metanit.com/sql/sqlserver/1.4.php
